# ISRO-SSN-FORK

## Client-Side Instructions
To setup client:

```sh
npm install
```
```sh
npm install @ionic/cli
```

To run client:

```sh
npm start
```

## Server-Side Instructions
To setup server:
```sh
npm install
```
```sh
npm i -g nodemon
```
```sh
pip install geopy==2.3.0
```
```sh 
pip install netCDF4==1.6.2
```
```sh
pip install numpy==1.24.1
```
```sh
pip install HeapDict==1.0.1
```

To start server:
```sh
nodemon app.js
```
